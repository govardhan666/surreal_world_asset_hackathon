# Extension Icons

This directory should contain the extension icons in the following sizes:
- icon16.png (16x16)
- icon32.png (32x32)
- icon48.png (48x48)
- icon128.png (128x128)

## How to create icons

1. Design a shield icon representing IP protection
2. Export in the required sizes
3. Place the PNG files in this directory

## Temporary Solution

For development, you can use emoji or create simple colored squares:
- Use tools like Figma, Canva, or GIMP
- Or use online tools like realfavicongenerator.net
